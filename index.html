Básico sobre a rede de internet

O que é DNS:

O DNS é como a agenda telefônica da internet. Ele traduz nomes de domínio (como google.com) para os endereços IP correspondentes (como 142.250.190.46), que são os números que os computadores usam para se comunicar e encontrar os sites na rede.

O que é IP:

O IP é o endereço de identificação de dispositivos conectados a uma rede.

O que é um domínio:

O domínio é o endereço que as pessoas usam para encontrar um site na internet, por exemplo: Philco.com.br. TLD é a parte final que indica o tipo ou a origem do site (.com.br, .com, .net, .org, etc.).

O que é URL:

A URL é o endereço completo que você digita no navegador para acessar um recurso específico na internet, como uma página web, imagem ou vídeo. Ela inclui o protocolo (http:// ou https://), o domínio (como www.exemplo.com) e o caminho para o recurso (como /pagina1.html).

O que é um servidor:

Um servidor é um computador ou sistema que fornece serviços, dados ou recursos para outros computadores, conhecidos como clientes, através de uma rede. Ele armazena, processa e gerencia informações, permitindo que os usuários acessem sites, aplicativos e arquivos remotamente.

O que é HTML

Inicio

O HTML não é uma linguagem de programação, mas sim uma linguagem de marcação. Sua única função é estruturar o conteúdo da web, sem ter a lógica de uma linguagem de programação (como variáveis ou laços).

  • Utiliza tags para organizar e apresentar conteúdo em páginas da web.
  • Não inclui funcionalidades como variáveis, loops ou condicionais, típicas de linguagens de programação.

Utilização das tags header, main e footer:

As tags <header>, <main> e <footer> são elementos semânticos em HTML5 que ajudam a estruturar o conteúdo de uma página web de forma clara e organizada.

  • <header>: Representa o cabeçalho da página ou de uma seção. Geralmente contém o título, logotipo, menu de navegação e outros elementos introdutórios.

  • <main>: Representa o conteúdo principal da página. Deve conter o conteúdo único e central, excluindo cabeçalhos, rodapés e barras laterais.

  • <footer>: Representa o rodapé da página ou de uma seção. Geralmente contém informações de contato, direitos autorais, links adicionais e outros detalhes finais.

Conceitos das tags:

Tags estruturais:

  1. <html>: Define o início e o fim do documento HTML.
  2. <head>: Contém metadados, como título e links para CSS.
  3. <body>: Contém o conteúdo visível da página.
  4. <header>, <main>, <footer>: Estruturas semânticas para cabeçalho, conteúdo principal e rodapé

Tags de texto:

  • <h1> a <h6>: Títulos de diferentes níveis.
  • <p>: Parágrafos de texto.
  • <strong>: Texto em negrito.
  • <em>: Texto em itálico.
  • <br>: Quebra de linha.
  • <hr>: Linha horizontal para separar conteúdo.

Tags de mídia:

  • <img>: Insere imagens.
  • <video>: Insere vídeos.
  • <audio>: Insere áudio.
  • <iframe>: Incorpora conteúdo de outras páginas.

Tags de links e listas:

  • <a>: Cria links para outras páginas ou recursos.
  • <ul>: Lista não ordenada (com marcadores).
  • <ol>: Lista ordenada (numerada).
  • <li>: Itens de lista.

Tags de formulários:

  • <form>: Define um formulário.
  • <input>: Campos de entrada (texto, senha, etc.).
  • <textarea>: Área de texto multilinha.
  • <button>: Botões clicáveis.

Tags de tabelas:

  • <table>: Define uma tabela.
  • <tr>: Define uma linha na tabela.
  • <td>: Define uma célula de dados.
  • <th>: Define uma célula de cabeçalho.

Sintex basica:

Uma tag de abertura, seguida pelo conteúdo, e depois uma tag de fechamento. Exemplo: <p>Este é um parágrafo.</p>.
Algumas tags são auto-fechadas, como <img src="imagem.jpg" alt="Descrição">.

Tipos de comentarios:

Comentários em HTML são inseridos entre <!-- e -->. Exemplo: <!-- Este é um comentário -->.
Comentários não são exibidos no navegador e são usados para explicar o código ou deixar notas para outros desenvolvedores.

Tipos de cabeçalho:

  • <h1>: Título principal, o mais importante.
  • <h2>: Subtítulo, menos importante que <h1>.
  • <h3>: Subtítulo de terceiro nível.
  • <h4>: Subtítulo de quarto nível.
  • <h5>: Subtítulo de quinto nível.
  • <h6>: Subtítulo de sexto nível, o menos importante.

O que é tag semânticas:

Tags semânticas são elementos HTML que descrevem claramente o significado do conteúdo que envolvem. Elas ajudam a estruturar a página de forma lógica e facilitam a compreensão tanto para desenvolvedores quanto para mecanismos de busca e tecnologias assistivas.
Exemplos de tags semânticas incluem <header> (cabeçalho da página), <nav> (navegação), <main> (conteúdo principal), <article> (artigo independente), <section> (seção do documento), <aside> (conteúdo complementar) e <footer> (rodapé).

Por que usar tag semânticas:

  • Melhoram a acessibilidade, facilitando a navegação para usuários com deficiências;
  • Ajudam os motores de busca a entenderem melhor o conteúdo da página, melhorando o SEO;
  • Tornam o código mais legível e fácil de manter para desenvolvedores;
  • Promovem boas práticas de desenvolvimento web, alinhando-se aos padrões da web;

Diferença entre tag semânticas e não semânticas:

  • Tags semânticas, como <header>, <article> e <footer>, fornecem significado ao conteúdo que envolvem, indicando sua função na estrutura da página. Elas ajudam na organização lógica do conteúdo e melhoram a acessibilidade e SEO;
  • Tags não semânticas, como <div> e <span>, não fornecem nenhum significado específico sobre o conteúdo. Elas são usadas principalmente para fins de estilo ou agrupamento de elementos, sem indicar a função ou importância do conteúdo dentro da página.

Inserção de imagem:

A tag <img> é usada para inserir imagens em uma página HTML. Ela é uma tag auto-fechada, o que significa que não requer uma tag de fechamento separada. A tag <img> utiliza atributos para definir a fonte da imagem, texto alternativo e outras propriedades.
Exemplo de uso da tag <img>:

  • src : Especifica o caminho para o arquivo de imagem.
  • alt: Fornece uma descrição alternativa da imagem, importante para acessibilidade.
  • width: Definem a largura e altura da imagem em pixels (opcional).

Inserção de vídeo:

A tag <video> é usada para incorporar vídeos em uma página HTML. Ela permite que você defina várias fontes de vídeo, controles de reprodução e outras propriedades para melhorar a experiência do usuário.
Exemplo de uso da tag <video>:

  • width e height: Definem a largura e altura do vídeo em pixels.
  • alt: Adiciona controles de reprodução (play, pause, volume, etc.).
  • width: Define as diferentes fontes de vídeo, permitindo compatibilidade com vários formatos.

Inserção de áudio:

A tag <audio> é usada para incorporar arquivos de áudio em uma página HTML. Ela permite que você defina várias fontes de áudio, controles de reprodução e outras propriedades para melhorar a experiência do usuário.
Exemplo de uso da tag <audio>:

  • controls: Adiciona controles de reprodução (play, pause, volume, etc.).
  • <source: Define as diferentes fontes de áudio, permitindo compatibilidade com vários formatos.

Tag style:

A tag <style> é usada para definir estilos CSS diretamente dentro de um documento HTML. Ela é colocada dentro da seção <head> do documento e permite que você escreva regras de estilo que serão aplicadas aos elementos da página.
